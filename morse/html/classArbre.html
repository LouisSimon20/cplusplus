Description détaillée

Classe définissant un arbre binaire permettant de passer du morse (.–) au francais.

Cette arbre permet de traduire le morse vers le francais.
L'arbre se base sur celui présent sur le site : https://radiobxi.org/blog/graphique-de-decodage-du-code-morse/
