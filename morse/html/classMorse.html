◆ tailleMessage()

int Morse::tailleMessage ( )
inline

Calcule la taille du message en ti.

Cette fonction permet d'écrire l'entête avant les données.
En effet, la taille du fichier est à spécifier dans l'entête.
La taille est calculée en ti : si le message est ".-", la taille sera de (1+1) + (3+1)

Renvoie
int - taille du message en ti
