bool Wav::detect_white ( int  pos)
inline

utilisé dans la fonction decoupage()

Permet de détecter le passage d'un symbole (ti ou taah) à un blanc
4 zeros consécutifs dans un symbole est un phénomène extrêmement rare : il caractérise donc le passage d'un symbole à un blanc.

Paramètres
positérateur du vecteur data
Renvoie
true - On vient de passer à un blanc
false - On est dans un symbole
